# GAME PLAYING CHESS

## Folder structures

-   ChessMain: handle user interaction, input, render board
-   ChessEngine: handle rules, valid move, log move, change state

## How to run

Excute command:

```
python3 ChessMain.py <mode> <first-turn>
```

`mode`: interger

-   0: agent vs random agent
-   1: person vs agent

`first-turn`: interger

-   0: person have the first turn
-   1: otherwise
